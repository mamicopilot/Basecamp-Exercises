# Claude Code ワークショップ

Claude Code を使って、稼働中のファクトリー在庫管理システム（Vue 3 フロントエンドと FastAPI バックエンドによる本格的なフルスタック構成）を探索・修正・拡張します。このエクササイズは自由形式です。唯一の正解はありません。自分では書いたことのないコードベースにおいて、Claude Code を開発パートナーとして体験することが目的です。

## ステップバイステップ ガイド

- **ライブサイト（推奨）:** https://claude-code-workshop.netlify.app/ — 任意の名前とワークショップコードを入力してください
- **オフライン用バックアップ:** コンパニオンサイトが利用できない場合は、このフォルダ内の [`workshop-guide.pdf`](./workshop-guide.pdf) を開いて完全なステップバイステップ手順を参照してください。

このエクササイズは 12 のコアステップとエキスパートチャレンジで構成されています。
